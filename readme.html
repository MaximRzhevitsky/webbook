1: Створено необхідні поля Advanced Custom Fields для товару. Ці поля включають додаткові зображення для товару. 2: Створено сторінку, "Сорінка усіх товарів" на якій відображаються всі товари, які присутні в магазині WooCommerce. Для відображення товарів використано loop WooCommerce та ACF-функції для отримання інформації про товари. 3: Замінено виведення мініатюр на виведення зображень, які вказано в полях ACF. Використано стандартні функції ACF для виведення зображень. 4: Додано можливість збільшувати зображення товару при наведенні на нього мишкою та відображення ціни товару на збільшеному зображенні. Використано jQuery для створення цієї функції. Так ож для виведення ціни товару використано стандартні функції WooCommerce. 5: Додано Ajax-функціонал для завантаження товарів без перезавантаження сторінки.
